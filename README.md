#### ⚪ Getting Strated
* ##### Prerequisites: npm, node
* ##### Installation & Execution
```bash
git clone https://github.com/MpqM/NestJS_Auth.git
cd {project}
npm install
# development
npm run start
# watch mode
npm run start:dev
# production mode
npm run start:prod
# unit tests
npm run test
# e2e tests
npm run test:e2e
# test coverage
npm run test:cov
```
